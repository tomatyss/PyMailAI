Installation
-----------

You can install pymailai using pip:

.. code-block:: bash

   pip install pymailai

For development installation with additional dependencies:

.. code-block:: bash

   pip install pymailai[dev]

For AI provider specific installations:

.. code-block:: bash

   pip install pymailai[openai]    # For OpenAI support
   pip install pymailai[anthropic] # For Anthropic support
   pip install pymailai[all]       # For all AI providers
